Demonstration App for LaunchDarkly

This demonstration app queries the public Parks On The Air API for "spots" showing amateur radio operators currently advertising a park operation, and displays them based on the selected mode.

You can filter by mode (e.g. SSB, CW, FT8) and/or by the listed entities (leading string match).

About POTA Spots

Parks on the Air (POTA) is a popular amateur radio program that encourages operators to "activate" state and national parks, promoting outdoor operation and public awareness of amateur radio.

Given the broad activity in the program, there should be some spots at most times of the day, especially US daytime and evening.

If no spots are shown, consider removing any mode or location filter

Usage

Request "http://.../spots/" to retrieve an unfiltered list of all current POTA spots.

Optionally:

Filter by Modes
Use .../mode/FT8,CW to filter by mode. Multiple modes can be specified.
Filter by Locations
Use .../locs/US-CA,US-MA to filter by location. Multiple locations can be specified.
Combined Filters
Use .../mode/SSB/locs/US-,CA- to filter by both mode and location.
User Name
To specify a user name, add a request parameter like "...?user=UserName". This works with or without filters in the path.
